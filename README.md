#내일배움캠프 일정 관리 과제

* 과제명 : 일정을 관리하는 앱 만들기
* 개발 일정 : 1/23 ~ 2/4
* 구현 단계 : 2

  ## 구현
  * 필수
    * Lv 0
       - API 명세서 작성
       - ERD 작성
       - SQL 작성
    * Lv 1
       - 일정 생성
       - 전체 일정 조회
       - 선택 일정 조회
    * Lv 2
       - 선택 일정 수정
       - 선택 일정 삭제
     
  * 도전
    * Lv 3 - 연관 관계 설정 [작성자외 일정의 연결]
    * Lv 4 - 페이지네이션
    * Lv 5 - 예외 발생 처리
    * Lv 6 - null체크 및 특정 패턴에 대한 검증
